LoCoBot WX200 Thesis

Human-Guided Autonomous Mobile Manipulator in Dynamic Environments: A Case Study Using LoCoBot

Project Overview

Welcome to the LoCoBot WX200 project! This initiative focuses on developing a human-guided framework for an autonomous mobile manipulator. The LoCoBot WX200 is designed for research applications and demonstrates capabilities like human-following, robotic arm control, and gesture-based human interaction.

Key Features

  • Gesture recognition for intuitive robot control
  • Human-following using depth camera-based navigation
  • Obstacle avoidance and autonomous navigation
  • Robotic arm manipulation for interaction tasks
